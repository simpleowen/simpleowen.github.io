Shell 是 Unix-Like 操作系统家族中被频繁使用的人机接口

比较常见的 Shell 有以下几种:

  • sh
  • ksh
  • csh
  • tcsh
  • bash
  • zsh

Shell 配置

Shell 配置文件可以分为系统级别的配置文件和用户级别的配置文件

系统级别的配置文件位于 /etc 目录下

用户级别的配置文件位于 ~ 目录下

在 Shell 启动时，会首先执行系统级别的配置文件(如果存在的话), 再执行用户级别的配置文件. 也就是说 ~/.bashrc 中的配置会覆盖 /etc/bashrc 中的配置.
