在 Slack 中订阅 Feeds

21, Jun, 2019
@damao

经常打开 Slack, 如果能将 feeds 也集成到 Slack, 就能减少一些 APP 切换成本.

前提

  • Slack 已登录
  • 已知 Feed URL

集成

Slack 为 WorkSpace 提供了 RSS APP, 可以订阅各类 feeds. RSS 会发送订阅内容到配置的 channel 或 user.

所以开始集成前, 你需要做一个小小的规划:

  • 是否新建一个 workspace 来接收订阅内容?
  • 使用 channel 来接收还是直接推送到 user?

确定好以后, 在 workspace 界面单击 Add apps, 输入 RSS 检索相关的 APP, 找到 RSS APP 后, 单击 INSTALL 将 APP 添加到 Slack.

此后, 就可以将 Feed URL 添加到 RSS APP 中, 并设置 Post to Channel

这样就可以在 Slack 对话窗口中接收 Feeds 了.
