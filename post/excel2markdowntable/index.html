Markdown Table Maker

Typora

Typora 是一款 What You See Is What You Mean 的 Markdown 编辑器.

它的界面非常清爽, 几乎没有多余的东西, 就像进入了写作沉浸模式.

Excel/GoogleSheet 格子转换成 Markdown Table 这种任务也能轻松完成, 只需将 Excel/GoogleSheet 格子粘贴到 Typora 中就能看到 Markdown Table 的预览效果, 简直惊艳.
